# 🧮 Basic Calculator

A simple and responsive **Basic Calculator** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean, modern, and user-friendly interface.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🖥️ Real-time display
- ⌨️ Keyboard support
- 🗑️ Clear (C) button
- ⌫ Backspace/Delete functionality
- ⚠️ Error handling for invalid expressions
- 📱 Fully responsive design

---

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 How to Use

1. Enter numbers using the buttons or keyboard.
2. Select an arithmetic operator.
3. Press **=** or **Enter** to calculate.
4. Use **C** to clear the display.
5. Use **⌫** or **Backspace** to delete the last character.
